# AQ///bimmer Tool - Official Releases

> BMW Diagnostic & Coding Tool for Windows

## Download

Download the latest release from the [Releases](https://github.com/tiraUnderCode23/AQudsTool-releases/releases) page.

## Features

- **BMW Vehicle Diagnostics** - Read/Clear DTCs, ECU identification
- **CAFD Coding** - FDL editing, One-Click coding, Cheat Sheets
- **Tool32** - EDIABAS job execution with PRG browser
- **FA/SVT Management** - Vehicle Order editing and comparison
- **Security Access** - 30+ BMW security algorithms
- **Car Control** - Windows, mirrors, lighting control
- **Live Data** - Real-time sensor monitoring
- **Arabic/English UI** - Full bilingual interface

## System Requirements

- Windows 10/11 (x64)
- BMW F/G/I Series vehicle with ethernet OBD adapter
- EDIABAS installation (optional, for advanced features)

## Installation

1. Download the latest `.zip` from [Releases](https://github.com/tiraUnderCode23/AQudsTool-releases/releases)
2. Extract to any folder
3. Run `aq_uds_tools.exe`
